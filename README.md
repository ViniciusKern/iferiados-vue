# iferiados-vue

Um guia completo de feriados brasileiros, implementado em [Vue.js](https://vuejs.org/).

[Live Demo](http://vkapps.com.br/vue/iferiados)


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
